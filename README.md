# Fancam Studio v2 🎥

飯拍 + 運動個人追蹤影片生成。兩段式 offline pipeline，設計目標：準備度 ≥98%。

## 準確度點做到 98%+
單靠即時 tracker 遮擋/交叉走位必然會跳 ID，所以 v2 用專業級流程：

1. **Pass 1**：BoT-SORT + ReID + 鏡頭移動補償（GMC），逐格追蹤全部人，儲低所有 tracklets 同全身外觀特徵
2. **離線縫合**：斷開嘅 tracklets 用「時間唔重疊 + 位置接近 + 外觀相似」自動縫返做同一個人
3. **目標鎖定**：
   - 🎤 飯拍：上載 1–3 張參考相，InsightFace 逐個 group 認樣，自動鎖定
   - ⚽ 運動：球員成日背向鏡頭，靠面唔穩陣 → 直接喺畫面撳個人，跟錯就去嗰秒再撳一下（keyframe 修正）
4. **準備度評分**：逐格計信心，報埋邊幾秒要覆核 — 98% 唔夠就話你知修邊度，修完再計，直至滿意先出片
5. 追蹤結果會 cache，改比例/解像度/zoom 重出**唔使重新分析**

## 輸出（自由選擇）
- 比例：9:16 / 16:9 / 4:5 / 1:1 / 3:4 / 任意自訂 W:H
- 解像度：720p / 1080p / 1440p / 4K / 任意自訂高度（闊度按比例自動計）
- H.264 CRF17 + 原聲 AAC

## 安裝 & 執行
```bash
pip install -r requirements.txt   # 另需 ffmpeg
python app.py                     # http://127.0.0.1:7860
```
GPU 加速：裝 CUDA 版 PyTorch + `onnxruntime-gpu`（4K 長片強烈建議）

## 流程
1️⃣ 上載 → 分析全片（揀模型：n 快 / m 平衡 / x 最準）
2️⃣ 揀目標（飯拍 tab 上載參考相自動認；運動 tab 直接撳畫面）
3️⃣ 計算準備度 → 覆核低信心時段（拉時間軸 → 撳目標 → 再計）
4️⃣ 揀比例/解像度 → 輸出
